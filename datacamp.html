Datacamp

C'est Quoi ?

Datacamp est une plateforme spécialisée 100% sur la Data et l'IA, axée sur la pratique du code, connu pour son interface de codage interactive dans le navigateur (pas besoin d'installer de logiciels). Datacamp vise les étudiants et data scientists qui veulent "mettre les mains dans le cambouis".

Gratuit ou Payant

Payant, abonnement annuel autour de 150€-300€, avec les premiers chapitres souvent gratuits.

Type de cours

Vidéos très courtes suivies immediatement d'exercices de code, manipulation de données (SQL, Python, R), Machine Learning, Deep Learning, IA Générative (LLM) allant du niveau débutant à intermédiaire. En bref, c'est le meilleur outil pour acquérir une fluidité de codage rapide en Python ou R.
